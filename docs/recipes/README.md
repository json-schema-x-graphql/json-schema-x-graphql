# JSON Schema to GraphQL Recipes

This directory contains common patterns and recipes for modeling GraphQL concepts using JSON Schema and `x-graphql` extensions.

## Available Recipes

- [**Polymorphism & Interfaces**](polymorphism.md): How to define Interfaces and implement them in Types.
- [**Field Arguments**](arguments.md): How to add arguments to fields.
- [**Federation**](federation.md): How to define Federated Entities with keys and directives.
- [**Unions**](unions.md): How to define Union types explicitly.
- [**Pagination (Relay)**](pagination.md): How to define Relay-style connections.

## Usage

Each recipe provides a JSON Schema snippet that you can copy into your project.
All examples follow the `json-schema-x-graphql` valid extensions.
